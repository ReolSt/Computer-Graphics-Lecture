프로그램 시작시에 사각형 바깥에 삼각형 하나와,
사각형 안쪽에 삼각형 둘을 배치합니다.
안쪽에 있는 삼각형은 양쪽 벽과 충돌 체크를 하여
부딪히면 방향을 바꿉니다.
바깥에 있는 삼각형은 바깥벽과 사각형 둘 다 충돌 체크를 하여
부딪히면 반대 방향으로 이동합니다.

마우스 클릭 시 삼각형이 추가되고, 최대 개수는 4개이며 
삼각형이 4개일 때 클릭하면 가장 먼저 추가된 삼각형이 삭제됩니다.
사각형 안쪽 클릭 시 안에 삼각형이 만들어지고, 바깥쪽 클릭시
바깥에 삼각형이 만들어집니다.
안쪽 삼각형은 좌우로만 움직이고 바깥쪽 삼각형은 삼각형 생성시에
정해지는 랜덤한 방향으로 움직입니다.

5, 6번과 마찬가지로 a를 누르면 내부가 채워진 삼각형,
b를 누르면 외곽선만 있는 삼각형을 그립니다.
