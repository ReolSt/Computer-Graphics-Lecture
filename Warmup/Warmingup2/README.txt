파일 읽기는 STL 라이브러리의 ifstream을 이용하였습니다.
강의에서 최대 80자까지라고 하셨는데 그냥 넉넉하게 버퍼 크기를 1024자로 수정하였고,

파일에서 한줄씩 End of File이 나올때까지 읽어 냅니다.
각 줄은 lines에 저장하여 처리가 끝난 후에 전체 내용을 출력할 수 있도록 합니다.

각 줄은, stringstream으로 가져와서 토큰화합니다.
cctype의 isdigit을 사용하여 토큰화한 각 단어에서 숫자가 있는지 파악하여,
숫자가 있으면 number_count의 값을 증가시키고, 없다면 word_coun의 값을 증가시킵니다.

파일의 끝까지 전부 읽은 뒤에,
파일의 전체 내용을 출력한 뒤,

word_count와 number_count를 출력합니다.
